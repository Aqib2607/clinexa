### üöÄ **About Clinexa**

Clinexa is a comprehensive, modern, and production-ready **Hospital Management System (HMS)** designed to streamline clinical workflows, automate laboratory operations, and manage hospital administration with high efficiency. It empowers healthcare providers with tools for real-time patient care, automated billing, and robust inventory management.

### üåü **Core Modules**

- **üè• Clinical & OPD**: Multi-step appointment booking, vital signs tracking, and digital prescriptions.
- **üõå Inpatient (IPD)**: Real-time ward management with graphical bed occupancy views.
- **üî¨ Lab Automation**: Machine integration via ASTM/HL7 and barcode-based sample tracking.
- **üíä Pharmacy & Inventory**: POS system for dispensing medicine and multi-store stock tracking.
- **üí∞ Finance & Accounts**: Centralized billing, multi-ledger accounting, and financial reporting.
- **üõ°Ô∏è Access Control**: Multi-tenant RBAC (Super Admin, Doctor, Nurse, Patient).

### ‚öôÔ∏è **Installation & Setup**

#### **1. Prerequisites**

- **Node.js**: v18+
- **PHP**: v8.2+
- **MySQL**: v8.0+

#### **2. Frontend Setup**

```bash
# Clone the repository
git clone https://github.com/Aqib2607/clinexa.git
cd clinexa

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

#### **3. Backend Setup**

```bash
# Navigate to backend folder
cd backend

# Install dependencies
composer install

# Environment configuration
cp .env.example .env
php artisan key:generate

# Database migration and seeding
php artisan migrate --seed

# Start the server
php artisan serve
```
